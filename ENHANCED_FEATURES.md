# HTTP客户端 - 增强版功能说明

## 🎉 新增功能概览

我们为HTTP客户端添加了两个重要功能：

### 1️⃣ **多标签页支持** 📑
- 支持同时打开多个请求标签页
- 每个标签页独立工作，互不干扰
- 轻松切换不同API测试
- 支持新建、关闭标签页

### 2️⃣ **历史记录管理** 📚
- 自动保存所有请求历史
- 支持自定义分类（如：工作、测试、开发）
- 快速搜索和检索
- 一键加载历史请求
- 支持导入/导出请求记录

## 🚀 使用指南

### 启动增强版客户端

```bash
# 启动增强版（多标签页 + 历史记录）
python3 http_client_enhanced.py
```

### 多标签页功能

#### 创建新标签页
- **方法1**: 点击工具栏"新建请求"按钮
- **方法2**: 使用快捷键 `Ctrl+N`
- **方法3**: 右键标签栏选择"新建"

#### 标签页操作
- **发送请求**: 点击"发送"按钮或按 `Ctrl+Enter`
- **保存请求**: 点击"保存"按钮或按 `Ctrl+S`
- **切换标签页**: 直接点击标签页标题
- **关闭标签页**: 点击标签页上的关闭按钮

#### 每个标签页包含
- ✅ URL输入框
- ✅ HTTP方法选择（下拉菜单）
- ✅ Headers编辑表格
- ✅ 请求体编辑区域
- ✅ 响应内容显示（Body + Headers）
- ✅ 响应状态/时间/大小统计

### 历史记录功能

#### 保存请求
1. 在标签页中输入请求信息
2. 点击"保存"按钮
3. 输入请求名称
4. 选择或输入分类
5. 点击确定保存

#### 查看历史记录
- **侧边栏**: 窗口右侧显示历史记录面板
- **分类筛选**: 从下拉菜单选择分类查看
- **搜索**: 在搜索框输入关键词实时搜索

#### 加载历史记录
1. 在右侧历史面板中查找请求
2. **双击**记录即可加载到当前标签页
3. 或先选择记录再点击"加载"按钮

#### 管理历史记录
- **删除**: 选择记录后点击"删除"按钮
- **刷新**: 点击"刷新"按钮更新列表
- **导出**: 菜单栏 → 文件 → 导出请求
- **导入**: 菜单栏 → 文件 → 导入请求

## 💡 使用技巧

### 1. 多标签页工作流
```
场景: 测试API的多个端点

步骤:
1. 点击"新建请求"创建标签页1
2. 输入GET请求 → 保存到"测试"分类
3. 再次点击"新建请求"创建标签页2
4. 输入POST请求 → 保存到"测试"分类
5. 标签页3: PUT请求
6. 标签页4: DELETE请求
7. 可以同时查看多个请求结果
```

### 2. 历史记录管理
```
场景: 组织API测试请求

步骤:
1. 为不同项目创建分类:
   - 项目A (API v1)
   - 项目B (API v2)
   - 开发测试
   - 生产验证

2. 保存请求时选择相应分类

3. 通过分类快速筛选:
   菜单 → 选择分类 → 查看所有相关请求

4. 搜索功能:
   在搜索框输入"get" → 快速找到所有GET请求
```

### 3. 快捷键
- `Ctrl+N` - 新建请求标签页
- `Ctrl+S` - 保存当前请求
- `Ctrl+Enter` - 发送当前请求

## 🔧 技术实现

### 数据库设计

#### requests表
```sql
CREATE TABLE requests (
    id INTEGER PRIMARY KEY,           -- 请求ID
    name TEXT NOT NULL,               -- 请求名称
    category TEXT,                    -- 分类
    method TEXT NOT NULL,             -- HTTP方法
    url TEXT NOT NULL,                -- 请求URL
    headers TEXT,                     -- 请求头(JSON)
    body TEXT,                        -- 请求体
    response_info TEXT,               -- 响应信息
    created_at TIMESTAMP,             -- 创建时间
    updated_at TIMESTAMP              -- 更新时间
)
```

#### categories表
```sql
CREATE TABLE categories (
    id INTEGER PRIMARY KEY,           -- 分类ID
    name TEXT UNIQUE NOT NULL,        -- 分类名称
    color TEXT,                       -- 分类颜色
    created_at TIMESTAMP              -- 创建时间
)
```

### 文件结构
```
http_client_enhanced.py    - 增强版主程序 (GUI + 多标签页)
history_db.py              - 历史记录数据库模块
test_enhanced_features.py  - 功能测试脚本
```
